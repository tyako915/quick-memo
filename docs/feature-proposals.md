# QuickMemo 機能改善提案

**作成日**: 2026-05-13  
**現在のバージョン**: MVP v1.0

---

## 現状の評価

| 項目 | 評価 |
|------|------|
| クロスデバイス共有 | ✅ URL共有で即時アクセス可能 |
| メモ管理 | ✅ 作成・編集・削除・ゴミ箱対応 |
| 同期 | ⚠️ 3秒ポーリング（遅延あり） |
| モバイル対応 | ⚠️ サイドバーが小さい画面で狭い |
| 検索・整理 | ❌ 未対応 |
| セキュリティ | ⚠️ URLを知れば誰でもアクセス可能 |

---

## 提案一覧

### 優先度：高（使い勝手の直接改善）

#### 1. 自動保存
**概要**: 入力を止めて1〜2秒後に自動でサーバーに保存する  
**理由**: 現在は「保存」ボタンを手動で押す必要があり、保存忘れのリスクがある  
**実装難易度**: 低（`useDebounce` で実装可能）

#### 2. メモ検索
**概要**: サイドバー上部に検索ボックスを設置し、タイトル・本文をリアルタイムフィルタリング  
**理由**: メモが増えると目的のメモを探しにくくなる  
**実装難易度**: 低（クライアントサイドのフィルタリングのみで可能）

#### 3. モバイル対応の強化
**概要**: スマートフォンでサイドバーをドロワー（スライドイン）形式に変更  
**理由**: 現在の横並びレイアウトはスマホで操作しにくい  
**実装難易度**: 中

#### 4. ゴミ箱の自動完全削除
**概要**: ゴミ箱に入れてから30日後に自動で完全削除するCronジョブを設置  
**理由**: ゴミ箱が溜まり続けるのを防ぐ  
**実装難易度**: 低（Vercel Cron + API Route）

---

### 優先度：中（快適性の向上）

#### 5. マークダウン対応
**概要**: 本文をマークダウン記法で記述でき、プレビューモードで整形表示できる  
**理由**: コードやリストを含むメモが読みやすくなる  
**実装難易度**: 中（`react-markdown` + `remark-gfm` で実装可能）

#### 6. メモのピン留め
**概要**: 重要なメモをサイドバーの最上部に固定表示できる  
**理由**: よく使うメモへのアクセスが速くなる  
**実装難易度**: 低（`pinned Boolean` カラム追加）

#### 7. リアルタイム同期（WebSocket / SSE）
**概要**: 3秒ポーリングをServer-Sent Events（SSE）に置き換え、即時反映を実現  
**理由**: 現在は最大3秒の遅延があり、複数デバイスで同時編集すると気になる  
**実装難易度**: 中

#### 8. メモのエクスポート
**概要**: 全メモをまとめてMarkdownファイルまたはJSONでダウンロードできる  
**理由**: バックアップや他ツールへの移行に対応できる  
**実装難易度**: 低（クライアントサイドのBlob生成）

---

### 優先度：低（将来的な拡張）

#### 9. アクセスパスワードの設定
**概要**: スペースにパスワードを設定し、知っている人だけがアクセスできるようにする  
**理由**: 現在はURLを知れば誰でもアクセス・編集できるため、意図しない第三者のアクセスを防げない  
**実装難易度**: 中（パスワードハッシュ + Cookieセッション）

#### 10. タグ・カテゴリ
**概要**: メモにタグを付けてフィルタリングできる  
**理由**: メモの種類（仕事・プライベート・コードなど）で整理できる  
**実装難易度**: 中（`Tag` テーブル追加 + UI改修）

#### 11. 複数スペースの管理画面
**概要**: 自分が作成したスペースの一覧を管理できる画面  
**理由**: 複数のスペースを用途別に使い分けたい場合に便利  
**実装難易度**: 高（軽量な認証が必要）

#### 12. 文字数・行数カウント
**概要**: エディタ下部に現在の文字数・行数をリアルタイム表示  
**理由**: テキストの量を把握しやすくなる  
**実装難易度**: 低

## 技術的な改善点（非機能）

| 項目 | 現状 | 改善案 |
|------|------|--------|
| ポーリング間隔 | 3秒固定 | タブがフォーカス時のみ実行（`visibilitychange`） |
| エラーハンドリング | なし | API失敗時にトースト通知を表示 |
| ローディング状態 | なし | メモ一覧取得中にスケルトンUIを表示 |
| SEO | デフォルト | OGP設定（タイトル・説明文） |
