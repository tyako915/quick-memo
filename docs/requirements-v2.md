## 1. プロジェクト概要

MVP v1.0 にて以下の機能が実装済み。

| ゴミ箱（ソフトデリート・復元） | ✅ 実装済み |
| クリップボードコピー | ✅ 実装済み |
| サイドバーによるメモ一覧表示 | ✅ 実装済み |
| 3秒ポーリングによるクロスデバイス同期 | ✅ 実装済み（SSEへ移行済み） |
| 自動保存・メモ検索・ゴミ箱自動削除 | ✅ 実装済み |
| モバイル対応・Markdownプレビュー・ピン留め | ✅ 実装済み |

### 対象ユーザー
複数デバイス（Windows / Mac / Linux / スマートフォン）を使い分けている個人ユーザー。

## 3. 機能要件

### Phase 2：即時着手可能（使い勝手の直接改善）

#### 3.1 自動保存
- [x] 本文・タイトルの入力が止まってから **1.5秒後** に自動でサーバーへ保存する
- [x] 自動保存中はツールバーに「保存中...」インジケーターを表示する
- [x] 自動保存完了後は「保存済み」を1秒間表示して消える
- [x] 手動の「保存」ボタンも引き続き利用可能とする
- [x] 実装方式：`useDebounce` フック（1500ms）

#### 3.2 メモ検索
- [x] サイドバー上部に検索ボックスを設置する
- [x] タイトル・本文をリアルタイムでフィルタリングする（クライアントサイド）
- [x] 検索ワードが入力されている間はゴミ箱タブの検索も対象とする
- [x] 検索ボックスをクリア（×ボタン）すると全件表示に戻る
- [x] 検索結果が0件の場合「一致するメモがありません」と表示する

#### 3.3 ゴミ箱の自動完全削除
- [x] ゴミ箱に移動してから **30日後** に自動で完全削除する
- [x] Vercel Cron Job（毎日0時）で `/api/cron/cleanup` を呼び出す
- [x] 削除対象：`deletedAt < NOW() - 30 days` のレコード
- [x] ゴミ箱一覧に「X日後に完全削除」と残日数を表示する

---

### Phase 3：UX改善

#### 3.4 モバイル対応強化
- [x] 画面幅 768px 未満でサイドバーを非表示にする
- [x] ハンバーガーメニューボタン（ヘッダー左）でサイドバーをドロワー表示する
- [x] ドロワー外をタップすると閉じる
- [x] メモ選択後にドロワーが自動で閉じ、エディタが全画面表示される

#### 3.5 マークダウン対応
- [x] エディタ右上に「編集 / プレビュー」切り替えタブを設置する
- [x] プレビューモードでは `react-markdown` + `remark-gfm` でレンダリングする
- [x] 対応記法：見出し・太字・斜体・コード・コードブロック・リスト・テーブル・リンク
- [x] 保存するデータはマークダウン記法の生テキストとする（変換しない）

#### 3.6 メモのピン留め
- [x] メモカードに「ピン」アイコンボタンを追加する
- [x] ピン留めしたメモはサイドバー一覧の先頭にまとめて表示する
- [x] ピン留め中はアイコンをハイライト表示する
- [x] ピン留め状態はDBに保存する（`pinned Boolean @default(false)`）

---







#### 3.9 リアルタイム同期（SSE）
- [x] 3秒ポーリングを Server-Sent Events（SSE）に置き換える
- [x] `/api/events?roomId=xxx` エンドポイントでSSEストリームを配信する
- [x] メモの作成・更新・削除をイベントとして送信する
- [x] 接続が切れた場合は自動で再接続する（`EventSource` の標準動作）

## 4. 非機能要件

| 項目 | 目標値 |
|------|--------|
| ページ初期表示 | 2秒以内（LCP） |
| 自動保存レスポンス | 500ms以内 |
| 検索フィルタリング | 入力から100ms以内に反映 |
| 可用性 | Vercel 標準 SLA に準拠 |
| モバイル対応 | iOS Safari / Android Chrome で動作確認済み |

## 5. データモデル変更点

```prisma
model Memo {
  id        String    @id @default(cuid())
  title     String    @default("")
  content   String
  roomId    String
  pinned    Boolean   @default(false)   // 3.6 ピン留め（Phase 3）
  createdAt DateTime  @default(now())
  updatedAt DateTime  @updatedAt
  deletedAt DateTime?

  @@index([roomId])
}

// 3.8 アクセスパスワード（Phase 4）
model Room {
  id           String    @id
  passwordHash String?
  createdAt    DateTime  @default(now())
}
```

## 7. 画面構成の変更

```
/                        → 共有メモスペースへリダイレクト（変更なし）
/room/[roomId]           → メモエディタ（サイドバー追加機能あり）
  ├─ サイドバー
  │   ├─ 検索ボックス          ← 新規（Phase 2）
  │   ├─ メモタブ（ピン留め順）← 変更（Phase 3）
  │   └─ ゴミ箱タブ（残日数）  ← 変更（Phase 2）
  ├─ エディタ
  │   ├─ 編集 / プレビュー切替 ← 新規（Phase 3）
  │   └─ 自動保存インジケーター← 新規（Phase 2）
  └─ ヘッダー
      └─ エクスポートボタン     ← 新規（Phase 4）
/room/[roomId]/auth      → パスワード入力画面（Phase 4）
```
